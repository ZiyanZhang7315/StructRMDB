API application program interface

This set of APIs was made to let our users reach the information they wanted in a new pragmatical way.

parameter description example
&species= *essential parameter. curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna' > Homo_sapiens_m6a_vienna.txt
&modification= *essential parameter.[m6A,PSI,ATOI] curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna' > Homo_sapiens_m6a_vienna.txt
&software= *essential parameter.[vienna,RNAstructure] curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna' > Homo_sapiens_m6a_vienna.txt
&gene_region= filters by the Gene Region, only for m7GDB [3' UTR, 5' UTR, Exon, Intron, Downstream, Distal] curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna&gene_region=Exon' > Homo_sapiens_m6a_vienna.txt
&StructRM_ID= filters by the StructRM ID. curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna&StructRM_ID=m6A_hg38_StruRM_vienna_521875' > Homo_sapiens_m6a_vienna.txt
&ID= filters by the ID. curl 'http://www.rnamd.org/StructRMDB/api.php?species=Homo_sapiens&modification=m6A&software=vienna&ID=m6A_hg38_10' > Homo_sapiens_m6a_vienna.txt


Download :
  • In data page, right-click or Ctrl+S could save data as json format.
  • The curl command can be used to download data directly in terminal.
  • We support up to 10000 pieces of data display and download. If you would like a larger dataset, please use Download.
